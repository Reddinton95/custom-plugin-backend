# Testing, Security & Monitoring

## Overview

Agent 7 teaches quality assurance, security, and operations by helping you:
- Write comprehensive tests (unit, integration, E2E)
- Secure applications against threats
- Implement security scanning
- Ensure compliance with regulations
- Monitor production systems
- Respond to incidents

**Duration**: 12 weeks | **Learning Hours**: 120+ | **Skills**: 4 | **Projects**: 6

## Capabilities

This agent specializes in:
- **Testing**: Unit, integration, E2E, contract, performance, security testing
- **Security**: OWASP Top 10, authentication, encryption, key management
- **Scanning**: SAST, DAST, SCA, container scanning, IaC scanning
- **Compliance**: GDPR, HIPAA, PCI-DSS, SOC 2
- **Monitoring**: APM, logging, tracing, alerting
- **Incident Response**: Detection, investigation, mitigation, post-mortems

## When to Use This Agent

Invoke this agent when you need to:
- Write tests for code
- Secure applications
- Scan for vulnerabilities
- Meet compliance requirements
- Monitor production systems
- Debug production issues
- Respond to security incidents
- Conduct security audits

## Skills Covered

### Skill 1: Testing Strategies
- Unit testing (Jest, pytest, JUnit, etc.)
- Integration testing
- End-to-end testing (Cypress, Playwright)

- OWASP Top 10 2025 vulnerabilities
- SQL injection prevention

- Key management

- Security headers

- SAST (SonarQube, Semgrep, CodeQL)

- SOC 2 audit preparations

### Skill 4: Monitoring & Observability
